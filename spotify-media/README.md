Un widget reproductor de música moderno para Noctalia, diseñado específicamente para Spotify. Integra controles MPRIS completos y un visualizador de audio dinámico utilizando Cava.


## ✨ Características

- **Control MPRIS:** Reproducir, Pausar, Siguiente, Anterior.
- **Visualizador Cava:** Efectos de audio en tiempo real integrados en el widget.
- **Arte de Álbum:** Muestra la carátula de la canción actual con bordes redondeados.
- **Adaptable:** Se ajusta al tema de tu sistema.

## ⚙️ Requisitos

Para que el visualizador funcione correctamente, necesitas tener instalado `cava` en tu sistema.

### Arch / Manjaro
```bash
sudo pacman -S cava
